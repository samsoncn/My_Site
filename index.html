One of the proudest projects I worked on was happened during this summer internship. It is a web photo editor for AI PoseRecognition model and I got to modify it to have more functionality as well as adjustments of the data outputs. It is not really a big project, but it stands as an important bridge for the data collection. I learned so much from playing and modifying with it, such as programming in python and java, and understanding of Coco dataset with data analysis skills. I also used Android Studio to modify an app for gesture detection testing. The purpose of the project is to allow users to upload and edit images, then the input function would convert information to a dataset for the AI model tuning and the final project, which is a fitness mirror that detect to do gestures. At the end of the project, I got a chance to teach a cooperation team to use the project to collect data. I was very glad that people were using my tools and I got to introduce my tool to them as I also learned what user experience is and I could do adjustments from feedback. This is one of the projects that I am very proud of.
